# InteractiveFiction
French Interactive Fiction called "Caradoc's Chronicles"

Le but est de créer un jeu vidéo textuel en utilisant le langage Prolog. Ce projet est un projet scolaire fait par des élèves de deuxièmes année de Bordeaux-INP - Ecole Nationale Supérieure de Cognitique.

Le repository est strucuré de la façon suivante :
- Le dossier "Doc" contient la documentation du projet ainsi que le rapport.
- Le dossier "Ressources" contient les ressources du projet à savoir un projet analogue et l'énoncé du projet
- Les 4 fichiers du code source de l'application sont placés à la racine du repository.
